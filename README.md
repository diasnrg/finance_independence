# Калькулятор финансовой независимости (FIRE Calculator)

Веб-приложение на Flutter для расчета финансовой независимости и раннего выхода на пенсию.

## Возможности

- Расчет целевой суммы для финансовой независимости по правилу 4%
- Учет инфляции при планировании
- Выбор инструмента инвестирования (индексный фонд, долларовый депозит)
- Интерактивный график зависимости ежемесячных накоплений от срока
- Визуализация структуры накоплений (вклады vs доход от процентов)

## Запуск проекта

```bash
flutter pub get
flutter run -d chrome
```

## Деплой

Проект автоматически деплоится на GitHub Pages при пуше в main ветку.
